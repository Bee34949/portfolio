เกี่ยวกับฉันเอง

สวัสดีครับ ผมชื่อ นายวงศพัทธ์ นาคม่วง นักพัฒนา Backend ที่มีความสนใจในด้านการจัดการระบบฐานข้อมูลและพัฒนา API ผมมีความมุ่งมั่นที่จะพัฒนาทักษะของตัวเองและสร้างสรรค์ผลงานที่มีคุณภาพ.

ปัจจุบันเป็นนักศึกษาชั้นปีที่ 3 สาขาวิทยาการคอมพิวเตอร์ มหาวิทยาลัยแม่โจ้.
