เกี่ยวกับฉัน

สวัสดีครับ ผมชื่อ นายวงศพัทธ์ นาคม่วง ปัจจุบันเป็นนักศึกษาชั้นปีที่ 3 ในสาขาวิทยาการคอมพิวเตอร์ มหาวิทยาลัยแม่โจ้ ผมมีความสนใจในด้าน Backend Development โดยเฉพาะการเขียนโปรแกรมด้วย Node.js และ Java เพื่อสร้าง API และระบบการจัดการข้อมูล

ตลอดระยะเวลาการเรียนและฝึกฝน ผมได้มีโอกาสพัฒนาทักษะทางด้านการพัฒนา Backend ผ่านโปรเจกต์ต่างๆ รวมถึงการสร้างระบบ เครื่องซักผ้าหยอดเหรียญ ที่เชื่อมต่อกับเว็บเพื่อการติดตามสถานะการใช้งานของเครื่องซักผ้า

ผมมีความมุ่งมั่นที่จะพัฒนาทักษะและประสบการณ์ในสายงานนี้ เพื่อสร้างสรรค์งานที่มีคุณภาพและประสิทธิภาพในการจัดการข้อมูลและระบบหลังบ้านที่มีความเสถียรและปลอดภัย
